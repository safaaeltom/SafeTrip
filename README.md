# 🚗 SafeTrip Landing Page

A modern, responsive landing page for a fictional travel service called **SafeTrip**. This project showcases a clean UI, structured layout, and strong call-to-action (CTA) strategy to guide users toward exploring and booking vehicles.


## 📌 Project Overview

SafeTrip is designed as a **single-page landing experience** that introduces users to a travel service and encourages them to explore available vehicles.

This project focuses on:

* Clean HTML structure
* Semantic layout (header, main, sections, footer)
* User experience (UX)
* Call-to-action (CTA) placement


## ✨ Features

* 🧭 Navigation with smooth scrolling
* 🏠 Hero section with main headline
* 💡 "Why Choose SafeTrip" section (benefits)
* 🚘 Car collection showcase with images
* ⭐ Customer testimonials
* 🔁 Multiple CTA buttons to guide user actions
* 📱 Responsive-ready structure
